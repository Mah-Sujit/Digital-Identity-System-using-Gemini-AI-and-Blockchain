# Digital Identity System using Gemini AI and Blockchain

## Overview

The Digital Identity System is a secure, decentralized identity management platform that combines Blockchain technology with Google Gemini AI to provide intelligent identity verification, document validation, fraud detection, and privacy-preserving authentication.

The platform enables users to create, manage, and verify digital identities while maintaining full ownership and control over their personal information. Blockchain ensures transparency and immutability, while Gemini AI enhances security through intelligent verification and analysis.

## Installation

### Clone Repository

```bash
git clone https://github.com/Mah-Sujit/Digital-Identity-System-using-Gemini-AI-and-Blockchain.git
```

### Navigate to Project

```bash
cd Digital-Identity-System-using-Gemini-AI-and-Blockchain
```

### Install Dependencies

```bash
npm install
```

### Start Backend

```bash
cd backend
npm install
npm start
```

### Start Frontend

```bash
cd frontend
npm install
npm start
```
